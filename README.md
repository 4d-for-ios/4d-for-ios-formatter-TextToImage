## Text to Image

* **Format:** Text ⟶ Image
* **Type:** basic formatter

## How to integrate

* To use a custom formatter, the first thing you'll need to do is create a YourDatabase.4dbase/Resources/formatters folder.
* Then drop the formatter folder into it.
